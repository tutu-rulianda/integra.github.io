Changelog

Stay updated with the latest improvements, features, and fixes in Integra Studio.

  Explore Changelog  
Nov 24, 2025

Version 3.1.0

Update
  • Conducted bug fixes.
  • Implemented enhancements to improve speed.
  • Revised the previously unsuccessful sideways face profile identification and verification process.
Oct 25, 2025

Version 3.0.0

Major Release

New Features

  • Introduced dashboards with features for identifying, verifying, predicting, recognizing, and detecting human faces.
  • Updated profile recognition system to detect faces by aligning and rotating profiles sideways.

Improvements

  • Enhanced face verification algorithm.
  • Enhanced gender recognition for individuals of diverse ethnicities.
  • Streamlined user interface to accommodate users of all ages and genders.
  • Discontinued using YOLO dependency for age and gender detection.
  • Bug fixes on enrollment process
Sep 25, 2025

Version 2.1.0

New Features
  • Multi-face detection has been implemented in group photos.
  • Enhanced age estimation algorithm with 15% improved accuracy.
  • Added support for profile face recognition
Sep 18, 2025

Version 2.0.5

Update
  • Optimized face matching algorithm for faster processing.
  • Improved user interface for mobile devices.
  • Enhanced security protocols for data protection.
Aug 10, 2025

Version 2.0.0

Major Release

New Features

  • Completely redesigned user interface with improved UX.
  • Added facial landmark detection for 68 key points.
  • Implemented emotion recognition with 7 basic emotions.
  • Utilized RDBMS for storing face profiles.
Jul 22, 2023

Version 1.5.3

Bug Fixes
  • Fixed issue with face detection in low-light conditions.
  • Resolved memory leak in face matching module.
  • Fixed gender misclassification for certain age groups.
  • Addressed security vulnerability in profile management.

Version 1.5.0

Update

  • Improved age estimation for children and elderly.
  • Enhanced face recognition with occlusion handling.
